Module 5 Data Collection via Excel

Learning Objectives

  • Understand how to collect ForestPlots data using Excel on a tablet

Set up your computer

In order to collect data on your tablet via excel, you need two things:

  1. To stay organised and
  2. Excel

Hopefully, you have already configured this in Module 2 and 3. If you have not set up your devices, return to the section Setting up your devices.

If you already have Excel on your tablet and laptop, then you are ready to prepare your field sheets.

Preparing field sheets on ForestPlots

If you already know how to prepare field sheets through the ForestPlots database, skip ahead to the next section.

If you need a refresher on how to create field sheets, watch this video or visit page 174 of the ForestPlots Manual.

If you’ve never been shown how to make field sheets before by a ForestPlots team member, reach out to the data officer of the country that you will be collecting data in and request training (see Module 2).

Preparing field sheets for the tablet

Preparing field sheets to collect data on a tablet is very similar to preparing regular field sheets. In fact, the first step is the same: - download a field sheet from ForestPlots,
- and format the field sheet to only see the columns that you want to work with.
